### ➡️
## starting a project

```
npm init -y
```

### Install Web Dev Server


#### Environment
install a web environment


```
npm i --save-dev @web/dev-server
```

add to scripts

```
"start": "web-dev-server --node-resolve --open --watch"
```

### Install Lit

```
npm install lit
```

# Let's Go
